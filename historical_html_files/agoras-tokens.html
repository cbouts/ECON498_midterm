Tau-Chain (or simply Tau) is a decentralized blockchain network under development with a governance mechanism that uses discussions and logic to create a dynamic system. Tau’s large-scale discussion platform can reportedly detect consensus points or disagreements between all users and execute or self-implement the network instructions. Agoras token will leverage Tau’s technology to have an economy governed by its users. Agoras tokens aim to enable markets for knowledge, computational resources, and derivatives.
